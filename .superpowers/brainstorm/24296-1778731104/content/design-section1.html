Architecture

The Brio theme lives inside apps/storefront-food. Theme selection is route-based:

  • /store/{slug} в†’ Classic food theme (orange, existing)
  • /store/{slug}/brio в†’ Brio theme (green/white)

Both themes share the same data layer (products, categories, cart, orders) but render with different UI components.

Shared vs Theme-Specific

SharedTheme-Specific
API calls, types, cart logicPage layouts, components, styling
Auth, checkout flowColors, typography, spacing
Product data, categoriesHomepage content, hero, story

Pros

  • Single app, no extra Docker/build pipeline
  • Shared cart, checkout, auth
  • Easy to add more themes later

Cons

  • Theme folder grows with each theme
  • Need theme-aware routing
