Build a professional corporate group website for "Al-Ektefa Group" using SvelteKit. 
Design inspiration: luxury refined aesthetic — dark navy + gold color palette, 
editorial typography, smooth page transitions. NOT a clone of any existing site.

=== TECH STACK ===
- SvelteKit with file-based routing
- Custom CSS only (no Tailwind, no UI libraries)
- Google Fonts: "Playfair Display" (headings) + "DM Sans" (body)
- Smooth fade/slide transitions between routes using SvelteKit page transitions
- Mobile responsive with hamburger menu
- TypeScript throughout (strict mode)
- Proper types for props, form data, and component interfaces

=== ANIMATIONS ===
- Page load: content fades in with staggered delay (CSS animation-delay)
- Scroll reveal: Intersection Observer on all sections — fade up on enter
- Stats counter: count up animation when stats section enters viewport
- Card hover: translateY + shadow
- Button hover: background shift + arrow moves right
- Route transitions: fade out/in between pages (SvelteKit onNavigate)

=== MOBILE RESPONSIVE ===
- Breakpoint: 768px
- Cards stack to single column
- Navbar collapses to hamburger
- Hero text scales down
- Form inputs full width
- Stats bar wraps to 2x2

=== IMPORTANT RULES ===
- All SVG icons must be inline (no external icon libraries)
- No placeholder "Lorem ipsum" — use real meaningful content throughout
- Form submit: preventDefault, show success toast/message, no backend
- All internal links use SvelteKit <a href=""> (not window.location)
- Gold color ONLY for: accents, icons, subtitles, CTA buttons, numbers — not body text
- Every page must have a sticky navbar and shared footer
- Smooth scroll for anchor links (#products, #contact, #features, #services)
